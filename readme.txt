== Description ==

Yahoo Finance Quotes is a wordpress plugin that allows you to manage and display your selected quotes (indices, stocks, etc) from yahoo finance on your wordpress blog.

== Installation ==

You can use the built in installer and upgrader, or you can install the plugin
manually.

1. You can either use the automatic plugin installer or your FTP program to upload it to your wp-content/plugins directory the top-level folder.
Don't just upload all the php files and put them in `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit your Yahoo Finance Quotes setting page in administrator area 
4. Add/Edit your desired quotes
5. That's it!

== Frequently Asked Questions ==

= How can I know symbols for yahoo quotes? =

Please visit http://finance.yahoo.com, there is an engine to help you find symbols.
